# Module Registry для AI-native Application Assembly Platform

В этом репозитории хранятся базовые переиспользуемые модули платформы.

## Структура

- `templates/` – шаблоны приложений и компонентов
- `black_boxes/` – готовые бинарные/сервисные модули
- `connectors/` – коннекторы к источникам данных и внешним системам
- `methodologies/` – архитектурные паттерны и методологии
- `test_packs/` – тестовые пакеты и шаблоны
- `specifications/` – переиспользуемые спецификации
- `policies/` – политики безопасности и комплаенс

Каждый модуль представлен папкой с файлом `manifest.yaml`, описывающим его свойства.
